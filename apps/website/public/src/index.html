A desktop and Android dictation app. Local mode runs Whisper AI on-device; cloud mode uses Groq Whisper for 99+ languages. Text is injected into whatever app has focus — no clipboard gymnastics, no app switching.

In local mode, yes — all audio is processed on-device via whisper.cpp. Nothing leaves your machine. Cloud mode sends audio over HTTPS only when you choose it. You're always in control of which mode is running.

Tap Get for Android to download the APK, then open it on your phone. You may need to allow installs from unknown sources — that's standard for apps distributed outside the Play Store.

Recording starts in under 10ms. Local transcription completes in under a second. Cloud mode via Groq Whisper is similarly fast. Long passages are chunked so nothing gets dropped mid-dictation.

Local mode supports English offline. Cloud mode unlocks 99+ languages with automatic detection — just speak and ColdVoice figures out the language.

Windows 10/11 desktop and Android. Both platforms share the same transcription and text-cleaning pipeline. The Windows app uses a global key hook for hotkeys; Android uses an accessibility service and IME.
