Questions

Common questions.

Does the installer include dictation support?

Yes. The Windows package is configured to include the local ASR executable and base English model.

Do I need an account to use desktop dictation?

The desktop app is local-first. Website accounts are wired for Supabase auth and future user flows.

Which platforms are being handled first?

This pass verifies Windows desktop and the public website. Mobile remains part of the wider goal, but it is not the current test target.
