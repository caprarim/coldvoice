# Frequently Asked Questions

### What is BridgeVoice?

A native desktop dictation app built with Tauri 2.0 and Rust. Local mode runs Whisper AI on-device; cloud mode uses Groq Whisper for 99+ languages. Text is injected into whatever app has focus.

### Is my voice data actually private?

In local mode, yes — all audio is processed on-device via `whisper.cpp`. No data leaves your machine. Cloud mode encrypts audio over HTTPS to the BridgeMind API. You choose.

### How do I get access to BridgeVoice?

Included with BridgeMind Pro ($50/month or $40/month annually). Pro unlocks BridgeVoice, BridgeMemory, BridgeMCP, BridgeSpace, premium skills, and the full prompt library.

### What platforms does it run on?

macOS, Windows, and Linux. Built with Tauri 2.0 and Rust. macOS builds are code-signed and notarized.

### Which Whisper models are available?

Local mode offers six models, from Tiny (75 MB) to Large-v3 (3.1 GB). Cloud mode uses Groq Whisper Large-v3-Turbo for 99+ languages.

### Do I need special hardware?

No. BridgeVoice works with any microphone, including built-in laptop mics. A quality headset provides the best experience.

### What languages does BridgeVoice support?

Local mode supports offline English. Cloud mode supports 99+ languages with automatic detection via Groq Whisper.
